About Convertly

Convertly is a fast and minimal unit converter app built for simplicity. It supports all major categories including length, weight, temperature, and more—all wrapped in a clean interface with zero ads or distractions.

Features

  • Clean, ad-free interface
  • Real-time conversion updates
  • Support for metric and imperial units
  • 100% offline — no data collected
  • Lightweight and responsive design
